# IA en Educación - Página Web

Esta página web presenta mi experiencia personal con la inteligencia artificial en el contexto educativo, reflexiones sobre su impacto y potencial futuro, además de recursos útiles y elementos interactivos.

## Características

- **Diseño Responsivo**: Se adapta perfectamente a dispositivos móviles, tablets y escritorio
- **Elementos Interactivos**: 
  - Íconos con animaciones
  - Carrusel de imágenes automático
  - Mini juego de trivia de IA
  - Navegación suave entre secciones
- **Paleta de Colores Personalizada**: Utilizando los colores del curso
- **Contenido Estructurado**: Dividido en secciones claras y fáciles de navegar

## Estructura del Proyecto

```
├── iaeneducacion.html      # Archivo principal HTML
├── styles.css             # Estilos CSS
├── script.js              # JavaScript para interactividad
├── images/                # Carpeta para imágenes
│   ├── ia-image-1.jpg
│   ├── ia-image-2.jpg
│   └── ia-image-3.jpg
├── README.md              # Este archivo
├── LICENSE                # Licencia MIT
└── .gitignore            # Archivos a ignorar en Git
```

## Secciones de la Página

1. **Experiencia Personal**: Descripción de mi experiencia con IA en educación
2. **Reflexiones**: Respuestas a preguntas clave sobre IA
3. **Enlaces Útiles**: Links a recursos importantes y páginas relacionadas
4. **Creatividad**: Proyectos creativos y elementos interactivos

## Tecnologías Utilizadas

- HTML5
- CSS3 (con variables CSS y grid/flexbox)
- JavaScript (ES6+)
- Font Awesome para íconos
- GitHub Pages para hosting

## Instalación y Uso

1. Clona este repositorio:
   ```bash
   git clone https://github.com/lmoralesdg/iaeneducacion.git
   ```

2. Navega al directorio del proyecto:
   ```bash
   cd iaeneducacion
   ```

3. Abre `iaeneducacion.html` en tu navegador o configura GitHub Pages

## Personalización

Para personalizar la página:
- Modifica `iaeneducacion.html` para cambiar el contenido
- Ajusta `styles.css` para modificar los estilos
- Actualiza `script.js` para añadir nueva funcionalidad
- Reemplaza las imágenes en la carpeta `images/`
